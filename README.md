# ModUpdateChecker

Discord bot that polls Mod providers for project updates and posts rich embeds to configured channels.

Current implemented mod providers are Modrinth thus far, however I plan to expand to CurseForge at a later time.

To install free, use this link from Discord:
https://discord.com/oauth2/authorize?client_id=1488025663171461120
